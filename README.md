## рҹӣ°пёҸ Overview

**TerraVia** is an end-to-end machine learning platform for satellite imagery analysis. Upload a `.tif` satellite image and get instant AI-powered insights вҖ” land use classification across 8+ categories and a real-time vegetation health index вҖ” all through a sleek, browser-based interface backed by a deployed deep learning model.
https://github.com/Samarthsawant/land-classification-and-vegetation-health/releases/download/v1/terravia_v2.html
> Manual land analysis used to take weeks. TerraVia does it in under 3 seconds.

---

## рҹҢҚ The Problem

| Challenge | Scale |
|-----------|-------|
| Global land cover has changed | **77%** of Earth's land area affected |
| Degraded land worldwide | **3.2 billion** hectares |
| Manual satellite analysis time | **~3 weeks** per region |
| Traditional methods | Expensive, unscalable, expert-dependent |

Remote sensing and land monitoring are critical for climate science, agriculture, urban planning, and disaster response вҖ” yet analysis at scale remains inaccessible without expensive software and domain expertise. TerraVia solves this.

---

## вңЁ Features

- рҹ—әпёҸ **Land Use Classification** вҖ” Classifies satellite imagery into 8+ land cover categories 
- рҹҢҝ **Vegetation Health Index** вҖ” Real-time NDVI-proxy score with stress classification (Healthy / Moderate Stress / High Stress)
- рҹ“Ҡ **Spectral Breakdown** вҖ” Top-3 class probability scores for every prediction
- рҹ–јпёҸ **Native TIFF Rendering** вҖ” Custom pure-JS TIFF parser with contrast stretching for 8-bit and 16-bit satellite imagery вҖ” no plugins needed
- вҡЎ **< 3 Second Inference** вҖ” Powered by a deployed Hugging Face Space backend
- рҹ“Ҙ **Export Results** вҖ” Download full predictions as JSON
- рҹ“ұ **Fully Responsive** вҖ” Works on desktop and mobile

## рҹ§  Model & Methodology

### Architecture
```
Raw Satellite Image (GeoTIFF)
         вҶ“
  Preprocessing & Normalization
         вҶ“
  Convolutional Neural Network (CNN)
  в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ
  в”Ӯ  Conv Blocks вҶ’ BatchNorm    в”Ӯ
  в”Ӯ  MaxPooling вҶ’ Dropout       в”Ӯ
  в”Ӯ  Dense Layers вҶ’ Softmax     в”Ӯ
  в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ
         вҶ“
  Multi-class Classification + Confidence Scores
         вҶ“
  Vegetation Health Computation
```

### Training Pipeline
1. **Data Ingestion** вҖ” EuroSAT + IndiaSAT imagery loaded and validated
2. **Preprocessing** вҖ” Normalization, augmentation (flips, rotations, spectral jitter)
3. **CNN Training** вҖ” Multi-class classification with cross-entropy loss
4. **Evaluation** вҖ” Per-class accuracy, F1, precision, recall on held-out test set
5. **Deployment** вҖ” Exported model served via Hugging Face Spaces API

## рҹҡҖ Getting Started

### Use the Live App
Visit the deployed TerraVia interface, upload any `.tif` satellite image, and get instant results.

> Make sure the Hugging Face Space backend is in **Running** state before analyzing.

### Run Locally

```bash
# follow the download link
https://github.com/Samarthsawant/land-classification-and-vegetation-health/releases/download/v1/terravia_v2.html

# Open the frontend
open terravia_v2.html
```

### Backend (Hugging Face Space)
The model is deployed at:
```
https://samdoesitbetter-terravia-api.hf.space/predict
```
Send a `POST` request with a `multipart/form-data` body containing your image file under the key `file`.

```python
import requests

with open("image.tif", "rb") as f:
    response = requests.post(
        "https://samdoesitbetter-terravia-api.hf.space/predict",
        files={"file": f}
    )

print(response.json())
# {
#   "label": "Forest",
#   "confidence": 97.1,
#   "vegetation_health": 84,
#   "all_scores": { "Forest": 97.1, "Agriculture": 1.8, ... }
# }
```
